# 转义字符

## 带有实体名称的 ASCII 实体

| 结果   | 描述             | 实体名称   | 实体编号  |
| ---- | -------------- | ------ | ----- |
| "    | quotation mark | &quot; | &#34; |
| '    | apostrophe     | &apos; | &#39; |
| &    | ampersand      | &amp;  | &#38; |
| <    | less-than      | &lt;   | &#60; |
| >    | greater-than   | &gt;   | &#62; |
